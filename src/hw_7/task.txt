Задание 7.1
Разработайте приложение электронной библиотеки. Основная задача приложения -
осуществление поиска книг по разным критериям. Приложение состоит из 3 классов: книга, библиотека, класс-запускалка.
Класс для запуска приложения должен находиться в пакете clientapp.
Классы книга и библиотека должны располагаться в пакете core.



1. Класс, описывающий книгу.
Выделите и реализуйте необходимые поля и/или методы.
Помимо этих полей дополнительно объявите свойства String[] последниеЧитатели и метод получитьПоследнихЧитателей,
возвращающий этот массив.
Но мы же не хотим утечек персональных данных? Поэтому - сделайте так, чтобы и это поле,
и этот метод - были доступны только в пакете core, и ни в коем случае не были доступны в классе для запуска приложения.



2. Класс, описывающий библиотеку, хранящую все имеющиеся в библиотеке книги (массив книг).
В этом классе должны быть нестатические методы:
- метод поиска книги по названию
- метод поиска книги по автору
- метод поиска книги по другому выбранному вами атрибуту книги



3. Класс-запускалка - имеет метод main, где инициализирует объект библиотеки с книгами и работает с функционалом
объекта библиотеки - вызывает методы поиска книг по разным критериям и выводит их на печать.
Метод печати книги должен быть определен в этом же классе, как статический,
позволяющий выводить в консоль все поля переданного в метод объект книги.
Если в метод был передан пустой (null) объект - вместо книги в консоль должно выводиться предупреждение.



В задании обязательно использование конструкторов с параметрами.
