Реализуйте генератор раздачи покерной карточной колоды (52 карты). Сгенерируйте саму колоду.
Карты не должны повторяться, все как в реальной жизни, двух бубновых тузов в колоде тоже не бывает. Перемешайте колоду.
Сделайте количество игроков и количество сдаваемых карт изменяемым,
т.е. чтобы код мог корректно реагировать на изменение соответствующих переменных.
По умолчанию раздача происходит на 2 человек по 6 карт.
В консоль выводить раздачу для каждого игрока.
Для вывода карточной масти использовать соответствующие unicode символы.
Карточные масти должны быть определены в enum - каждая константа должна ассоциировать с собой:
выводимый символ масти, короткое название (3 символа верхний регистр, пример - ПИК, БУБ), полное название.
Использование списков не требуется, будет достаточно массивов.


Fisher–Yates shuffle
Чтобы перемешать массив a из n элементов (индексы элементов от 0 до n - 1):
  для всех i от n − 1 до 1 выполнить
       j ← случайное число 0 ≤ j ≤ i
       обменять местами a[j] и a[i]


import java.util.Random;

public class Main {
    public static void main(String[] args) {
        int[] nums = {1, 2, 3, 4, 5, 6, 7, 8, 9};
        shuffleArray(nums);
        for (int i = 0; i < nums.length; i++) {
            System.out.print(nums[i] + " ");
        }
    }

    static void shuffleArray(int[] arr) {
        Random rnd = new Random();
        for (int i = arr.length - 1; i > 0; i--) {
            int index = rnd.nextInt(i + 1);
            int temp = arr[index];
            arr[index] = arr[i];
            arr[i] = temp;
        }
    }
}
